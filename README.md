## 1. Requirements

- Node.js (21.7.3)
- npm (latest)
- esno (latest)

## 2. Install and run

### Install

```bash
npm run setup
```

### Run (development)

```bash
npm run dev
```

## 3. Build and run

### Local

#### Build

```bash
npm run build
```

#### Run (production)

```bash
npm run start
```

## 4. Variables

See [`.env.template`](.env.template) for all the available variables.
